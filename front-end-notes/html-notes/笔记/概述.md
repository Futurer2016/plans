## 什么是 HTML ？
HTML: Hyper Text Markup Language, 超文本标记语言, 是W3C组织定义的语言标准，HTML是用于描述页面结构的语言。

HTML决定了页面有什么东西，该东西表示什么含义

## 什么是 CSS？
CSS: Cascading Style Sheets, 层叠样式表, 是W3C组织定义的语言标准，CSS是用于描述页面展示的语言。

CSS决定了页面长什么样子
