横向规则测试:

宽度 为 auto

宽度的默认值是auto: 将剩余空间吸收掉

margin 为 0

margin的默认值为0, 若width、padding、border、margin计算后，仍然有剩余空间，该剩余空间被margin-right全部吸收 （文档从右向左排列时，margin-left吸收剩余空间）

width 为auto 并且 margin 为 auto

width 为auto 并且 margin 为 auto. margin: auto; 表示将剩余空间吸收掉，其吸收能力小于宽度

margin 为 auto ，其他都定值

margin 为 auto ，其他都定值。margin吸收空间的能力被显示出来。 margin-left 与 margin-right 吸收能力相同，表现为居中

margin 为 负值 ，宽度默认

margin 为 负值 ，宽度默认, 宽度会将由于margin负值导致的过多剩余空间全部吸收掉
