探究行高的细则

行高的相对大小，对于本元素来说是相对于本元素的字体大小,即如果字体大小为12px,行高为2em, 则行高最后计算的结果为28px

但行高在父元素定义，并继承给后代元素时会出现一些问题，如下：

1. 固定行高

在文本过大时固定的行高会使文字发生重叠，文本小时文字距离又太大
