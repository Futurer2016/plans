1. z-index: -∞

    堆叠层级1

    创建堆叠上下文的元素的背景和边框

  2. z-index: < 0

    堆叠层级2

    堆叠级别为负值的堆叠上下文, 两个元素刻度相同时, 后面的覆盖前面的

  3. z-index: auto

    堆叠层级3

    常规流非定位的块盒

  4. z-index: auto

    堆叠层级4

    非定位的浮动盒子

  5. z-index: auto

    堆叠层级5

    常规流非定位行盒

  6. z-index: auto

    堆叠层级6

    任何 z-index 是 auto 的定位子元素，以及 z-index 是 0 的堆叠上下文

  7. z-index: > 0

    堆叠层级7

    堆叠级别为正值的堆叠上下文

每个堆叠上下文，独立于其他堆叠上下文，它们之间不能相互穿插。
